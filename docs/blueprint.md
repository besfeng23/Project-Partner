# **App Name**: Project Partner

## Core Features:

- Project Tracking: Track project details like name, status, and associated connectors.
- Task Management: Manage tasks with Kanban board: Backlog, Doing, and Done.
- Decision Logging: Log decisions with context and consequences (append-only).
- Artifact Storage: Store project artifacts like links, notes, and uploaded files.
- AI Project Partner: An AI tool to suggest next actions, create tasks/decisions/constraints, and identify risks, after it READS from Firestore.
- GitHub/Vercel Integration: Connect to GitHub and Vercel for metadata and optional webhooks.
- Memory Summarization: Summarize conversations into memorySummaries to persist AI learnings. Mark messages as writtenToMemory.

## Style Guidelines:

- Primary color: Deep Indigo (#6666FF), evoking trust and clarity.
- Background color: Dark charcoal grey (#1A1A1A) for a sophisticated, dark-themed interface.
- Accent color: Electric Purple (#BF40BF) for interactive elements and highlights.
- Headline font: 'Space Grotesk', a sans-serif for a techy feel; body text: 'Inter', a grotesque sans-serif suitable for both headlines and body text.
- Minimal, clean icons to represent project elements.
- Clean cards for project components with a focus on 'Recommended Next Action' prominence.
- Subtle animations on state changes and AI actions to maintain user engagement.
